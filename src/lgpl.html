Request for modified LGPL 3.0 License.

One of Apple's criteria for accepting a program is whether or not it makes calls to unsupported Apple API's. By requiring static linking, they can prove that the software does not make such calls.

Unfortunately this introduces an incompatibility with the go-ethereum LGPLv3 license. We request a modified LGPLv3 license that will allow static linking. Which will enable us to submit our application, Status - a mobile dapp browser, to the App Store and in return we will open source our entire client.
